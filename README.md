# Portfólio Pessoal - Norberto Diman Junior

Esta é uma página de perfil (Linktree style) desenvolvida como parte dos meus estudos no curso **Técnico em Informática no Senac Araraquara**. O projeto visa consolidar conhecimentos em estruturação HTML, estilização moderna com Tailwind CSS e integração de fontes e ícones externos.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas:

- **HTML5**: Estruturação semântica do conteúdo.
- **Tailwind CSS (v4)**: Framework utilitário para estilização rápida e responsiva.
- **Font Awesome**: Biblioteca de ícones para as redes sociais.
- **Google Fonts**: Integração da fonte *Montserrat*.
- **Catppuccin Mocha**: Paleta de cores de alto contraste e baixa fadiga visual.

## 🎨 Design

A página utiliza a paleta de cores **Catppuccin**, especificamente o tema **Mocha**, que oferece uma estética moderna e minimalista com tons pastéis sobre um fundo escuro.

| Elemento | Cor (Hex) |
| :--- | :--- |
| Fundo (Base) | `#1e1e2e` |
| Texto Principal | `#cdd6f4` |
| Accent (Links/Hover) | `#89b4fa` |
| Cards/Botões | `#181825` |
